# Almeida & Costa Advocacia

Site institucional estático para um escritório de advocacia, pronto para ficar guardado no GitHub e ser publicado depois.

## Arquivos

- `index.html`: página completa com HTML, CSS, JavaScript e imagem embutidos.

## Como abrir

Abra `index.html` diretamente no navegador. Não precisa instalar dependências.

## Antes de publicar

Troque os dados fictícios:

- nome do escritório;
- endereço;
- e-mail;
- telefone;
- áreas de atuação;
- nomes e descrições da equipe;
- destino real do formulário.
